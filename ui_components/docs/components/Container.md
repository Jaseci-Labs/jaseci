---
title: Container
---

#### Summary

A `Container` is a general-purpose component, analogous to a `div` element, that will render a `div` you can style and attach events to.

<u>Example:</u>

```JSON
{
	"component": "Container",
	"css": {
        "width": "200px",
        "height": "200px",
	}
}
```
