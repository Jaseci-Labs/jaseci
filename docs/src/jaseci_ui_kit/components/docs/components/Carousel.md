title: Carousel
---

#### Summary

Create an image slideshow

<u>Example:</u>

```JSON
{
  "component": "Carousel",
  "props": {
    "images": [
      {
        "src": "http://example.com/image.png"
      }
    ]
  }
}
```

### Props

| name   | type                   | description |
| ------ | ---------------------- | ----------- |
| images | `Array<{src: string}>` |             |
