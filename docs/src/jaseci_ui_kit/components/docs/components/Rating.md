#### Summary

Allow users to rate something and collect a discrete value

<u>Example:</u>

```JSON
{
    "component": "Rating",
    "props": {
      "stars": 10,
      "value": 2
    }
}
```

### Props

| name  | type     | description           |
| ----- | -------- | --------------------- |
| stars | `number` |                       |
| value | `2`      | Sets an initial value |
