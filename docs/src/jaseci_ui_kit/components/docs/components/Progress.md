---
title: Progress
---

#### Summary

Display a progress bar

<u>Example:</u>

```JSON
{
    "component": "Progress",
    "props": {
      "palette": "accent",
      "value": 20
    }
}
```

### Props

| name    | type                                                                                                       | description        |
| ------- | ---------------------------------------------------------------------------------------------------------- | ------------------ |
| palette | `primary` or `secondary` or `accent` or `info` or `warning` or `error` or `success` or `accent` or `ghost` |                    |
| value   | `number`                                                                                                   | A value from 1-100 |
