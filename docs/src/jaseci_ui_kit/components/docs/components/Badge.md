#### Summary

Simple badge/chip component

<u>Example:</u>

```JSON
{
      "component": "Badge",
      "props": {
        "name": "badge1",
        "label": "Project",
        "size": "md",
        "palette": "primary"
      }
}
```

### Props

| name    | type                                                                                | description |
| ------- | ----------------------------------------------------------------------------------- | ----------- |
| label   | `string`                                                                            |             |
| palette | `info` or `warning` or `primary` or `secondary` or `success` or `accent` or `ghost` |             |
| size    | `xs` or `sm` or `md` or `lg`                                                        |             |
