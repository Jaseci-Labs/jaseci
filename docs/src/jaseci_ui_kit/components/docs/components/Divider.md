#### Summary

Use to divide components

<u>Example:</u>

```JSON
{
    "component": "Divider",
    "props": {
        "label": "OR",
        "orientation": "vertical"
    }
}
```

### Props

| name        | type                        | description |
| ----------- | --------------------------- | ----------- |
| label       | `string`                    |             |
| orientation | `vertical` or `orientation` |             |
