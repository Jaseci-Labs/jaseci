title: Text
---

#### Summary

A `Text` component is a that renders elements in various typographic-styles.

<u>Example</u>

```JSON
{
	"component": "Text",
	"props": {
		"variant": "simple",
		"value": "Hello world!"
	}
}
```

#### Props

| name    | type                |
| ------- | ------------------- |
| variant | `simple` or `title` |
| value   | `string`            |
