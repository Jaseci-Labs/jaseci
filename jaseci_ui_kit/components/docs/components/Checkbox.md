---
title: Checkbox
---

#### Summary

Capture boolean input

<u>Example:</u>

```JSON

{
  "name": "checkbox1",
  "component": "Checkbox",
  "props": {
    "label": "Correct?",
    "palette": "{{config:fieldPalette}}",
    "size": "xs"
  }
}
```

### Props

| name    | type                                 | description |
| ------- | ------------------------------------ | ----------- |
| label   | `string`                             |             |
| size    | `xs` or `sm` or `md` or `lg`         |             |
| palette | `primary` or `secondary` or `accent` |             |
