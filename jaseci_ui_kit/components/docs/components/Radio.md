---
title: Radio
---

#### Summary

Displays a Radio button

<u>Example:</u>

```JSON
{
    "component": "Radio",
    "props": {
      "label": "Cat",
      "palette": "primary"
    }
}
```

### Props

| name    | type                                                                                                       | description |
| ------- | ---------------------------------------------------------------------------------------------------------- | ----------- |
| label   | `string`                                                                                                   |             |
| palette | `primary` or `secondary` or `accent` or `info` or `warning` or `error` or `success` or `accent` or `ghost` |             |
