#### Summary

General purpose card component

<u>Example:</u>

```JSON
{
  "component": "Card",
  "props": {
    "variant": "outline",
    "shadow": "md",
    "outlineColor": "red",
    "radius": "lg"
  },
  "css": {
    "padding": "20px 20px"
  },
  "sections": {
    "children": []
  }
}
```

### Props

| name    | type                           | description |
| ------- | ------------------------------ | ----------- |
| variant | `outline` or `shadow`          |             |
| shadow  | `sm` or `md` or `lg` or `xl`   |             |
| radius  | `sm` or `md` or `lg` or `full` |             |
